# 使用说明

第一次使用请只运行一个文件：

```text
UESTC_Setup_Wizard.exe
```

不要手动运行 `background_runner.exe`，它会由向导自动设置为后台开机自启动。

## 一步一步操作

1. 解压下载到的压缩包。
2. 打开解压后的文件夹。
3. 双击 `UESTC_Setup_Wizard.exe`。
4. 输入账号/学号。
5. 输入密码。
6. 选择认证页面地址：
   - 寝室公寓：`http://aaa.uestc.edu.cn`
   - 寝室公寓：`http://10.253.0.235`
   - 主楼教学楼：`http://10.253.0.237`
7. 选择 `ac_id`：
   - 主楼/教学楼一般选 `1`
   - 寝室/公寓一般选 `3`
8. 选择运营商：
   - 校园网：`@dx-uestc`
   - 电信：`@dx`
   - 移动：`@cmcc`
9. 其他选项不懂就保持默认。
10. 点击“保存配置并安装后台任务”。

看到“安装完成”后，就可以关闭向导窗口。

## 每个程序的作用

| 文件 | 作用 |
| --- | --- |
| `UESTC_Setup_Wizard.exe` | 配置向导，首次使用运行它 |
| `background_runner.exe` | 后台自动登录，不要手动点 |
| `login_once.exe` | 手动测试登录一次，一般不用 |
| `log_cleanup.exe` | 自动清理日志，不要手动点 |

## 日志在哪里

日志在 `logs` 文件夹里。

如果出错，请打开当天的日志文件，例如：

```text
logs\2026-05-12.log
```

## 重要提醒

配置完成后，`config.py` 里会保存你的账号和密码。

不要把配置后的文件夹发给别人。
